- **Database:** Supabase (coming soon)
- **Hosting:** Vercel (free tier)
- **Domain:** lovequest.vercel.app

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
